# Assessment-1-Programming-Skills-Portfolio
In partial fulfillment of the course Introduction to Programing, this portfolio is designed to showcase the skills that the student has learned throughout the semester. The exercises demonstrated in this portfolio is a mix of basic and advanced coding.

Exercise #1 Coding is Cool
Demonstrates how concatenate string using comma and plus operation

Exercise #2 Simple Sums
Demonstrates basic arithmetic operation by adding 2 numbers and displaying result 

Exercise #3 Biography
Collects personal information through user input then stores the data in dictionary and displays the collected information.

Exercise #4 Primitive Quiz

Exercise #5 Days of the Months
Determines the number of days in a given month. It checks for leap years and changes February's days from 28-29 when needed

Exercise #6 Brute Attack Force
Creates a password entry system that allows user to have 5 attempts to log in the correct password. After exceeding the maximum attempt the system alerts about multiple failed log in. 

Exercise #7 Some Counting
Demonstrates how to use for loops with in range() function.

Exercise #8 Simple Search
Demonstrates searching a name using a function. The user is prompted to enter the name they want to search for, and the function checks if the name exist in the list.

Exercise #9 Hello
Demonstrate how to use a function with a greeting Hello

Exercise #10 Is it even?
Creates a program that tests if the value is even or odd using functions.
